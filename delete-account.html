Suppression de Compte - OSFUT

Dernière mise à jour : 4 décembre 2025

⚠️ Attention : La suppression de votre compte est définitive et irréversible. Toutes vos données seront supprimées de manière permanente.

Comment supprimer votre compte OSFUT

Méthode 1 : Depuis l'application (Recommandé)

  1. Ouvrez l'application OSFUT sur votre appareil
  2. Connectez-vous à votre compte
  3. Allez dans votre profil (icône de profil en haut à droite)
  4. Cliquez sur "Paramètres" ou "Gérer mon compte"
  5. Sélectionnez "Supprimer mon compte"
  6. Confirmez la suppression en entrant votre mot de passe
  7. Votre compte et toutes vos données seront supprimés définitivement

Méthode 2 : Par email

Si vous ne pouvez pas accéder à l'application, vous pouvez demander la suppression de votre compte par email :

  1. Envoyez un email à : contact.polouxe.osfut@gmail.com
  2. Indiquez dans le sujet : "Demande de suppression de compte OSFUT"
  3. Précisez dans le corps de l'email :
    • Votre adresse email utilisée pour votre compte OSFUT
    • Votre nom d'utilisateur (si différent de l'email)
    • La confirmation que vous souhaitez supprimer définitivement votre compte
  4. Nous traiterons votre demande dans un délai de 7 jours ouvrés

Données supprimées

Lors de la suppression de votre compte, les données suivantes seront supprimées de manière permanente :

  • Données de profil : Nom, prénom, âge, genre
  • Données d'authentification : Email, mot de passe (déjà crypté)
  • Rooms : Toutes les rooms que vous avez créées
  • Événements : Tous les événements que vous avez créés
  • OSFUTs : Toutes vos propositions OSFUT
  • Messages : Tous vos messages de chat
  • Réservations privées : Toutes vos réservations de calendrier
  • Statuts de participation : Tous vos statuts aux événements
  • Tokens de notification : Tous vos tokens de notification push
  • Amis et interactions : Toutes vos relations et interactions

ℹ️ Note importante :

  • Les données supprimées ne peuvent pas être récupérées
  • Vous ne pourrez plus accéder à votre compte après suppression
  • Si vous êtes membre d'une room créée par quelqu'un d'autre, vous serez retiré de cette room, mais la room elle-même ne sera pas supprimée
  • Vos messages dans les rooms resteront visibles pour les autres membres (mais sans votre nom associé)

Délai de suppression

La suppression de votre compte est effectuée dans un délai de 30 jours après votre demande. Pendant cette période, vous pouvez annuler la suppression en vous reconnectant à votre compte.

Après 30 jours, toutes vos données seront définitivement supprimées de nos serveurs et ne pourront plus être récupérées.

Conservation de certaines données

Conformément aux obligations légales, certaines données peuvent être conservées plus longtemps si la loi l'exige :

  • Données nécessaires pour des raisons fiscales ou comptables
  • Données nécessaires pour résoudre des litiges
  • Données nécessaires pour respecter les obligations légales

Ces données seront conservées uniquement dans la mesure et pour la durée requise par la loi.
